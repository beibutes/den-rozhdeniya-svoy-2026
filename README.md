# Мой день рождения — приглашение

Одностраничный сайт-приглашение на день рождения (41 год).
**Когда:** 9 октября 2026, 18:00 · **Где:** ресторан Svoy, ул. Жумабаева, 24, Астана.

## Описание
Страница-приглашение для внешних гостей: информация о событии, живой таймер
обратного отсчёта до начала и интерактивный wish-list с возможностью брони
подарков. Забронированную позицию другие гости выбрать уже не могут.

## Возможности
- **Hero-приглашение** — дата, время, адрес, возраст.
- **Таймер обратного отсчёта** до 9 октября 2026, 18:00 (часовой пояс UTC+5).
- **Wish-list** — 10 позиций с фото и ценовым диапазоном в тенге (₸).
- **Бронирование подарков** — гость указывает имя; занятая позиция помечается
  «Забронировано» и блокируется для других. Бронь можно отменить.
- **Адаптивная вёрстка** — корректно на телефоне и десктопе.
- Фото с автоматическим SVG-fallback, если изображение не загрузилось.

## Запуск локально
Открыть `index.html` в браузере. Для корректной работы загрузки лучше через
локальный сервер:
```
cd projects/den-rozhdeniya-svoy-2026
python3 -m http.server 8000
# открыть http://localhost:8000
```

## ⚠️ Важно про бронирование (общий доступ из интернета)
Сейчас брони хранятся в **localStorage** браузера — это работает как демо, но
бронь видна только в том же браузере, а не у всех гостей. Для настоящей общей
брони между всеми посетителями нужен общий бэкенд.

В `js/booking.js` сделана абстракция `BookingStore` с тремя методами
(`getAll`, `reserve`, `cancel`). Чтобы включить общую бронь — реализовать эти
три метода поверх Supabase или Firebase и заменить `window.BookingStore`.
Остальной код менять не нужно.
